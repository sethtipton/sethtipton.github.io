# sethtipton.github.io

A personal portfolio built with Astro, TypeScript, Sass, and selective React islands.

## Scripts

```bash
npm install
npm run dev
npm run build
npm run test
```

## Deployment

GitHub Pages deployment is configured in
`.github/workflows/deploy.yml`. The Astro `site` and `base` values can
be overridden with `SITE_URL` and `BASE_PATH` environment variables so the
same scaffold can work for either a user site or a project site.
